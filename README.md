# Personal Portfolio Website

A modern, high-performance, and responsive portfolio website built with Next.js and Tailwind CSS. This project features custom UI elements, smooth animations, and a clean user experience.

## Live link: https://themefisher-nu.vercel.app


## Tech Stack
* **Framework:** Next.js 15+ (App Router)
* **Styling:** Tailwind CSS
* **Icons:** React Icons & Custom SVG Components
* **Deployment:** Vercel

## How to Run Locally

1. **Clone the repository:**
2. **Navigate to the project directory:**
  ```
  cd your-repo-name
  ```
3. **Install dependencies:**
  ```
  npm install
  ```
4. **Start the development server:**
  ```
  npm run dev
  ```
5. **Open in your browser:** Go to http://localhost:3000 to see the result.
